🧩 Структура проекта
'''
project/
├── data/                  # Входные данные 
├── models/                # Обученные ML-модели и графики по результатам
├── core/
│   ├── ml/                # Машинное обучение
│   │   ├── train_models.py     # Обучение моделей (XGBoost, RandomForest и др.)
│   │   └── predict.py          # Предсказания по сохранённым моделям
│   ├── data_loader.py     # Загрузка и валидация входных данных
│   └── physics.py         # Расчёт физико-химических характеристик (масса, ёмкость, потери и т.д.)
├── optimization/
│   └── genetic_optimizer.py  # Генетический алгоритм оптимизации параметров
├── visualization/
│   ├── plots/             # Папки с сохранёнными графиками
│   └── plot_utils.py      # Модули построения графиков, heatmap, 3D и анимаций
├── gui/                   # Модуль пользовательского интерфейса
└── main.py                # Основной скрипт запуска, связывает все этапы
'''

⚙️ Возможности:
    📈 Физическое моделирование: 
        расчёт ёмкости, энергии, КПД, саморазряда и др. по параметрам пористости, ESR, толщины, площади и т.п.

    🤖 Машинное обучение: 
        предсказание характеристик суперконденсатора по исходным признакам с помощью XGBoost, RandomForest, SVM, LinearRegression.

    🧬 Оптимизация: 
        многокритериальный генетический алгоритм с учётом ёмкости, срока службы, потерь, саморазряда и т.д.

    📊 Визуализация: 
        гистограммы, тепловые карты, 2D/3D графики, анимации заряд-разряд.

    🧪 Анализ параметров: 
        влияние материала, электролита, плотности тока, сканирования и др. на эффективность устройства.


🚀 Запуск:
    # Запуск GUI
    python main_gui.py

📂 Примеры файлов:
    
    1.Файл получаемый в результате оптимизации параметров (оптимизация.xlsx)

    2.Файл с расчитанными значениями и предсказанными параметрами в конце (Результаты_с_прогнозами.xlsx)

    3.Файлы визуализации в папке visualization
