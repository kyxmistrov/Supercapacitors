# 🧩 Supercapacitors Optimization & ML Project

Комплексная система для моделирования, предсказания и оптимизации параметров суперконденсаторов на основе физических моделей, машинного обучения и генетических алгоритмов.

---

## 📁 Структура проекта

```
project/
├── data/                  # Входные данные
├── models/               # Обученные ML-модели и результаты
│
├── core/
│   ├── ml/              # Машинное обучение
│   │   ├── train_models.py   # Обучение моделей (XGBoost, RF и др.)
│   │   └── predict.py        # Предсказания на основе моделей
│   │
│   ├── data_loader.py   # Загрузка и валидация данных
│   └── physics.py       # Физико-химические расчёты
│
├── optimization/
│   └── genetic_optimizer.py  # Генетический алгоритм оптимизации
│
├── visualization/
│   ├── plots/           # Сохранённые графики
│   └── plot_utils.py    # Построение графиков (2D, 3D, heatmap)
│
├── gui/                 # Пользовательский интерфейс
└── main.py              # Главный файл запуска системы
```

---

## ⚙️ Возможности проекта

### 📈 Физическое моделирование
- Расчёт:
  - ёмкости
  - энергии
  - КПД
  - саморазряда
- Учитываются параметры:
  - пористость
  - ESR
  - толщина электродов
  - площадь поверхности

---

### 🤖 Машинное обучение
- Предсказание характеристик суперконденсаторов
- Используемые модели:
  - XGBoost
  - RandomForest
  - SVM
  - Linear Regression

---

### 🧬 Оптимизация
- Генетический алгоритм
- Многокритериальная оптимизация:
  - ёмкость
  - срок службы
  - потери энергии
  - саморазряд

---

### 📊 Визуализация
- 2D / 3D графики
- Heatmap анализ
- Гистограммы распределений
- Анимации процессов заряд/разряд

---

### 🧪 Анализ параметров
- Влияние:
  - материала электродов
  - электролита
  - плотности тока
  - режима сканирования
- на эффективность устройства
